#Vrindha-Store-Data-Analysis(Interactive Dashboard creation using MS Excel)

The Vridha Store Sales Report is an interactive data analytics dashboard developed in Microsoft Excel to analyze and visualize the overall sales performance of Vridha Store across multiple business dimensions such as gender, age group, state, and sales channel.
This project aims to transform raw sales data into meaningful insights using data cleaning, pivot tables, charts, and slicers. The dashboard allows users to dynamically filter data and observe real-time changes in trends and key performance metrics.

Tools & Techniques Used

•	Microsoft Excel (Advanced Features):
o	Pivot Tables for dynamic data summarization
o	Pivot Charts for visual representation of KPIs
o	Slicers for interactive filtering by Gender, Age Group, and State
o	Data validation and formatting for accuracy and clarity
o	Chart types: Column chart, Bar chart, Pie chart, Line chart

•	Data Analysis Concepts:
o	Sales trend analysis (Monthly and Yearly comparison)
o	Channel performance evaluation
o	Demographic segmentation (Gender & Age Group)
o	Geographic sales analysis (Top States & Cities)
o	Order fulfillment tracking

This Dashboard give answer of some question which are as below:
1)	How do monthly orders and sales perform throughout the year?
2)	Which product categories contribute most to sales?
3)	What percentage of orders are delivered, returned, or cancelled?
4)	Which online platforms contribute the most to total sales?
5)	Which gender contributes more to sales revenue?
